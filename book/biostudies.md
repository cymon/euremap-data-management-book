# 7. BioStudies

The primary purpose of the BioStudies resource is to aggregate and provide access to
descriptions of biological studies, along with links to the data generated by
those studies, whether the data resides in other EMBL-EBI databases, external
repositories, or as additional data that doesn’t fit into existing structured
archives.

A single BioStudies accession should map to a single BioProject accession,
however multiple BioSamples and data repository accessions (e.g. ENA,
Metabolights, ArrayExpress) a be linked to a BioStudies accession.
