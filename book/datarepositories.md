# 6. Data Repositories

EUREMAP recommends the following data repositories:


- [*European Nucleotide Archive (ENA)*](https://www.ebi.ac.uk/ena/browser/home) - for sequence data, incl. metagenomics, transciptomics, etc

- [*BioSamples*](https://www.ebi.ac.uk/biosamples) - to register organismal samples and add sample specific metadata 

- [*ArrayExpress*](https://www.ebi.ac.uk/biostudies/arrayexpress) - microarray or qPCR data

- [*Metabolights*](https://www.ebi.ac.uk/metabolights) - for spectrographic data (LC-MS, UV-vis, etc)

- [*ChEMBL*](https://www.ebi.ac.uk/chembl) - molecular assay results, structures, etc…

- [*BioImage Archive*](https://www.ebi.ac.uk/bioimage-archive/) - for imaging data (see also BioStudies)

- [*BioStudies*](https://www.ebi.ac.uk/biostudies/) - for study metadata and result data in tabular format, with some scope to include images (depending on size). We suggest all organismal studies should include a BioStudies accession to record project metadata, and possibly to deposit result data

- [*Protocols.IO*](https://www.protocols.io) - for protocols/methods if these are an data output of the
  project. Submit as part of the [EUREMAP Protocols Community](https://www.protocols.io/workspaces/euremap-protocols-community).
