# 6. Data Repositories

    - BiozStudies - for study metadata and result data in tabular format, with some scope to include images (depending on size). We suggest all organismal studies should include a BioStudies accession to record project metadata, and possibly to deposit result data
    - BioSamples - to register organismal samples in the BioStudies accession and add sample specific metadata 
    - EMBL ENA - for sequence data, incl. metagenomics, transciptomics, etc
    - ArrayExpress - microarray or qPCR data
    - ChEMBL - molecular assay results, structures, etc…
    - Metabolights - for spectrographic data (LC-MS, UV-vis, etc)
    - Protocols.IO - for protocols if these are an output
