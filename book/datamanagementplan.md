# 2. Data Management Plan

The EUREMAP project uses the [OpenAIRE Argos](https://argos.openaire.eu/portal)
service to create and organise data management plans.

A DMP within Argos (called a *"Plan"*) consists of a *"Blueprint"* template of
generic information to which data set *"Descriptions"* are added as necessary to
describe all the data sets to be collected by the project.

**Procedure for creating a DMP in Argos:**

- Create an [account on Argos](https://argos.openaire.eu/portal) if necessary, and login.

- EUREMAP has a pre-formated project-specific *"Blueprint"*. To begin a new
  project Plan, click the *"Start New Plan"* button on the top-right of the
  frontpage. In the Search bar, search for *"EUREMAP Plan Blueprint"*, and
  select.

- Complete the fields in Sections 1 and 2.

- **For each data set (or data type) to be created**, select an appropriate
  "Description" in Section 3. The default should be "EUREMAP | generic | template
  Version 1". Complete the information required in the Description template, and
  repeat as necessary for each data set.

- On publishing your completed DMP, it will be accessioned within the Zenodo
  repository where it can be included in the [Zenodo EUREMAP
  Community](https://zenodo.org/communities/euremap/records?q=&l=list&p=1&s=10&sort=newest)
  space.

- Several examples of Public Plans using the EUREMAP templates can be found by
  searching in Argos and Zenodo, e.g. [EUREMAP demonstrator project: Enhancing
  the Production and Therapeutic Potential of Streptocyclinones for Alzheimer’s
  Disease](https://argos.openaire.eu/explore-plans/overview/public/71a0d90d-8b3e-4f8d-81c1-e1d0ef950385)
