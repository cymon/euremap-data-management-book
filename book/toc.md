# EUREMAP Project Data Management Book

A guide to how data are managed using the EUREMAP Marine Bioprospecting Pipeline

## Quick Start

### 1. [EMBL ENA Webin Submission Portal](./webinportal.md)
    Create an account in the ENA Webin Submission Portal. 

### 2. BioProject/Study
    Create a BioProject/Study accession from the Webin Portal

### 3. BioSamples
    Submit sample metadata to BioSamples through the Webin Portal

### 4. Submit your dataset to a data repository
    Submit your data the the appropriate data repository, referencing both the
    relevant BioSamples and BioProject accessions

### 5. BioStudies
    Create a BioStudies accession using the BioStudies Submission Tool
    Link BioProject, BioSamples, additional data files, and all data repository accessions to your BioStudies accession
